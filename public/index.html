Есть 3 страницы отдающие ответы: Главная, Статистика, Комментарии

Главная Статистика Коментарии

Главная

Доступна по адресу /handlers/v1/

принимает GET запросы. На любой запрос отвечает "Hello".

Не важно тело запроса.

Статистика

Доступна по адресу /handlers/v1/stats

Принимает GET запросы. Введёт статистку запросов.

Отправляет json вида {"user-agent": count_request}

Не важно тело запроса.

Комментарии

Доступна по адресу /handlers/v1/comments

Принимает POST запросы.

В теле запроса отправляем обычный текст, который будет записан в массив.

Отправляет полный массив со всеми комментариями.

Если в теле запроса ничего не будет, страница вернёт текущий массив комментраиев.
